# Course Platform Compatibility Guide

## Goal
Align The Dope Cloud Teacher courses with quality expectations commonly seen on Coursera, Pluralsight, and LinkedIn Learning.

## Baseline Structure for Every Course
- Clear course outcomes (3-7 measurable outcomes)
- Lesson-level objectives for each module
- Video lesson + practical lab for each lesson
- Knowledge checks (quiz or scenario review)
- Capstone or final applied project
- Completion criteria and certificate rules

## Content Standards
- Accuracy: align with official docs for Azure, AWS, GCP, and exam provider updates
- Clarity: plain-language explanation plus technical depth
- Relevance: tie each lesson to a real cloud role and task
- Recency: refresh links, versions, and screenshots quarterly
- Engagement: include examples, challenge prompts, and reflection questions

## Delivery Standards
- Video target: 6-10 minutes per lesson
- Captioning/transcript support for every video
- Downloadable resource per module (checklist, template, or cheat sheet)
- Lab guidance with setup, budget/safety notes, and expected output
- Assessment with answer rationale

## Assessment Standards
- Lesson checks: 3-5 questions minimum
- Module checks: scenario-based questions
- Final assessment: practical demonstration or capstone artifact
- Passing threshold: define and display per course

## Metadata Standards (Platform-Ready)
- Course title, subtitle, and target learner profile
- Level (Beginner/Intermediate/Advanced)
- Estimated duration and lesson count
- Skills map and role outcomes
- Certification alignment (where applicable)

## Recommended Review Cadence
- Monthly: verify links and external references
- Quarterly: update exam objectives, cloud service changes, and screenshots
- Biannual: rewrite weak-performing lessons based on completion and engagement data

## Ready-to-Ship Checklist
- [ ] Learning outcomes visible
- [ ] Lesson objectives present
- [ ] Lab instructions tested
- [ ] Quiz keys validated
- [ ] Capstone rubric included
- [ ] Transcript/captions provided
- [ ] Completion certificate logic confirmed
- [ ] Analytics events firing for enrollment, lesson opens, and completions
