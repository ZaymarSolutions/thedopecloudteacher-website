PG Parks Cloud & AI Instructor Guide

Schedule: 6 weeks, 2 days per week, 90 minutes per session. This guide is printable for in-class facilitation and substitute coverage.

Session rhythm

Use workshop timing for small groups: 10 min opener, 15 min concept bursts, 45 min guided build, 15 min Shark Tank pitch, 5 min exit ticket.

12-session run of show

Session 1 (W1 Day A)

  • Innovation Lab: Build It With AI workshop
  • Create business idea, brand, flyer, social post, and pitch
  • Exit ticket: one thing AI sped up in your workflow

Session 2 (W1 Day B)

  • Service models quick lesson
  • Finish Cloud Life Map
  • Share 30-second project pitch

Session 3 (W2 Day A)

  • Prompt engineering fundamentals
  • Prompt remix challenge
  • Save version history

Session 4 (W2 Day B)

  • Responsible AI checkpoints
  • Build study-helper prompts
  • Peer test and improve

Session 5 (W3 Day A)

  • Data basics and chart types
  • Clean sample dataset
  • Create first chart

Session 6 (W3 Day B)

  • Data story framing
  • Build dashboard snapshot
  • Add one insight sentence

Session 7 (W4 Day A)

  • Security basics: MFA + phishing
  • Security checklist sprint
  • Risk scoring exercise

Session 8 (W4 Day B)

  • Cloud identity mini-lesson
  • Create secure-account guide
  • Team review and revise

Session 9 (W5 Day A)

  • Portfolio purpose and examples
  • Build one-page portfolio shell
  • Add about section

Session 10 (W5 Day B)

  • Artifact curation
  • Add project screenshots and links
  • Peer accessibility check

Session 11 (W6 Day A)

  • Career and certification roadmap
  • Demo rehearsal rounds
  • Feedback and final edits

Session 12 (W6 Day B)

  • Final showcase presentations
  • 90-day action plan card
  • Celebrate + next-step commitments

Fast differentiation for mixed ages

Core win: complete baseline task. Pro move: add one advanced feature. Pair students so one leads setup and one leads explanation, then switch roles. If only a few learners attend, run as a workshop conversation instead of a lecture.

Session 1 workshop checkpoints (90 minutes)

Checkpoint 1

  • Icebreaker: If you had $500, what business would you start?
  • Define AI and cloud with real student examples
  • Start Cloud Quest scoring board

Checkpoint 2

  • Run business idea prompt in ChatGPT
  • Pick one idea and generate brand package
  • Refine slogan and mission with learner voice

Checkpoint 3

  • Generate flyer and social media content
  • Rework copy for target audience clarity
  • Practice 60-second pitch structure

Checkpoint 4

  • Shark Tank style presentations
  • Debrief AI, cloud, prompting, marketing, entrepreneurship
  • Assign homework and preview website build day
