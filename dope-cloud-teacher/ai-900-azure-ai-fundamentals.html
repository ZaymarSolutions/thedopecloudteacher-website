Course Curriculum

Six modules for an AI-900 foundation

This structure follows the current Microsoft AI fundamentals flow and gives learners a clean path into Azure AI services, vocabulary, and exam readiness.

Module 1: AI Workloads and Responsible AI Foundations

Topics

  • What AI workloads look like in business
  • Common terms: ML, NLP, vision, conversational AI, generative AI
  • Responsible AI principles and risk awareness
Module 2: Machine Learning Basics Core concepts

Topics

  • Training data, models, and predictions
  • Regression, classification, clustering basics
  • Azure Machine Learning overview
Module 3: Computer Vision Image intelligence

Topics

  • Image analysis and object detection
  • OCR and document extraction
  • Vision use cases inside Azure AI services
Module 4: Natural Language Processing Language AI

Topics

  • Sentiment, key phrase extraction, classification, translation
  • Language Studio and text analytics services
  • Question answering and conversational interfaces
Module 5: Generative AI on Azure Modern AI layer

Topics

  • Foundations of large language models
  • Azure OpenAI concepts, prompts, and grounding
  • Enterprise safety and retrieval-aware design
Module 6: Search, Bots, and Applied AI Architecture Putting it together

Topics

  • Azure AI Search and intelligent retrieval
  • Bot concepts and human-in-the-loop support
  • Choosing the right Azure AI service for the use case
